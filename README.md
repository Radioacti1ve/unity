# unity
Программирование игр

Механики:
1) Переключение между двумя состояниями мира: реальностью и миром духов. При смене состояния изменяется доступность объектов уровня, что позволяет по-разному взаимодействовать с одним и тем же пространством и использовать особенности каждого мира для прохождения.
2) В мире духов появляется клон, повторяющий последнюю записанную последовательность действий игрока из реального мира. Клон становится частью головоломки: может активировать кнопки, удерживать триггеры и помогать открывать путь дальше.
